The following projects uses these files and libraries:

- os. : used for python efficiency and interacting with other modules
- cs50
    - SQL : used to access .db files and execute functions using SQL
- flask
    - Flask : to create the web application
    - redirect : go to another web function
    - render_template : go to a webpage in templates
    - request: request data and input from html forms
    - session: used for individual user activity
- flask_session
    - Session: used for individual user activity
- werkzeug.security
    - check_password_hash: check if the password input matches the hash generated by the function
    - generate_password_hash: generate a hash of the user password for confidentiality
